Name Assignment

Description:

Using scratch to give my name actions and effects. All of them required action and there must be at least three different ones. The program runs when green flag is clicked and stops and goes back to beginning stage when stop sign is clicked.

Concepts Learned:

  • Green flag, when ___ key is pressed: conditional statements.
  • Modifying sprite and background: stylize the character and settings as I wish.
  • Forever loop: to continue a set of actions forever.

Computer Buying Project

Description:

Using scratch to give a presentation on four computers of choice. There must be a conversation between two sprites and images of the computer showing. Provide information about the computers in your presentation and include the final decision.

Concepts Learned:

  • Wait __ seconds: modifying the timing of appearance and disappearance of sprites
  • Ghost effect: making the sprite look semi transparent or fully gone. Used in order to hide computer sprites.
  • If ___ is pressed, then: Using a key as the condition statement and let the project flow on by pressing different keys.

Final Project

Desciption:

Create a game or story (I chose game so game description from now on) that includes an introduction/help explaining how to play, different levels, signal for win, and creativity. It is consisted of an intro where SAM (a sprite) introduces the game to the player and the player can control Gerald the sea lion to collect shells. The game can go on as long as you want since it is an infinity loop.

Concepts Learned

  • Broadcast: have sprite perform basic actions after a signal. I used this for my intro.
  • More blocks: use a series of steps to define a personalized block therefore saving space
  • Operators: use math and variables to set a condition. In the game I used it to control the shells falling from the sky, the score, and the time.
